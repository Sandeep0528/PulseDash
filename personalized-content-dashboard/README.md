# PulseDash | Personalized Content Dashboard

A high-performance, responsive dashboard built with **Next.js 14**, **Redux Toolkit**, and **Tailwind CSS**. It features a unified feed of real-time news and movies with drag-and-drop capabilities.

## 🚀 Features
- **Unified Feed:** Real-time data from NewsAPI and TMDB (Movies).
- **Personalized Search:** Search across multiple sources with debouncing logic.
- **Drag & Drop:** Customize your layout using `@dnd-kit`.
- **Favorites System:** Save items for later using Redux state persistence.
- **Dark Mode:** Fully responsive UI with light/dark theme support.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **State Management:** Redux Toolkit & RTK Query
- **Styling:** Tailwind CSS & Lucide Icons
- **Drag & Drop:** DND-Kit
